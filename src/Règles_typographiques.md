Règles Typographiques
=====================
Les règles typographiques à suivre sont les suivantes :
- **les espaces** : les ponctuations doubles (`? ! ; :`) ainsi que les guillemets français (`« »`) sont associés à une espace _insécable fine_ ;
- **les tirets** : (demi-)cadratins sont acollé aux mots qu’il précède ou suit _sans espace insécable_ (par exemple —comme ça— c’est très bien) ;
- **ergonomie clavier** — faire comprendre de quelle touche et de quel caractère on parle
  - **les touches de claviers** physique sont indiqué avec les balises `<kbd>F</kbd>`, ici pour indiquer la touche marquée « F » en _Qwerty_ ;
  - **les caractères produit** sont indéqué dans des balises `<code>n</code>`, (ou leur équivalent markdown : `\`` quand c’est possible) ;
- **les emphases**
  - l’utilisation de l’italique se fait de préférence via les symboles `_` (ex. `_italique_`) ;
  - l’utilisation du gras se fait de préférence via les symboles `**` (ex. `**gras**`) ;

Toute contribution est encouragée à suivre les règles ci-dessus.
